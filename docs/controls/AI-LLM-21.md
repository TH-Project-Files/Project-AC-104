# AI-LLM-21: Install data loss prevention (DLP) tools to stop AI agents from accidentally or maliciously leaking sensitive company data.

## Control Details
Detailed Description:
Data Loss Prevention (DLP) tools are security solutions that identify, monitor, and protect sensitive information—such as intellectual property, PII, and financial records—from unauthorized access or transmission. In the context of AI agents, DLP involves deploying specialized software that inspects the data flowing into prompts (inputs) and the information generated by the models (outputs) to ensure that confidential company data is not leaked to external AI providers or unauthorized users.

Why AI Compounds Risk:
AI exacerbates data leakage risks because agents often operate with high levels of autonomy and can move data through multi-hop chains, including vector stores and downstream automations, which traditional security controls cannot see. Furthermore, techniques like Retrieval-Augmented Generation (RAG) can inadvertently strip away original document access controls, while prompt injection attacks can manipulate agents into revealing sensitive training data or internal documents that should remain private.

Examples:
1. Deploy an AI Gateway, such as an enterprise API Gateway, to serve as a single point of inspection for all API-based AI traffic, allowing the organization to enforce real-time content monitoring and block sensitive data from leaving the network perimeter during LLM interactions.
2. Use browser-native security extensions, like an enterprise browser security extension or a browser security solution, to provide last-mile visibility and granular control over AI sessions, enabling the organization to block copy-paste or file upload actions within specific generative AI tools on both managed and unmanaged devices.
3. Implement semantic-aware DLP solutions, such as semantic-aware DLP or AI-driven data security posture management (DSPM) solutions, which use machine learning to understand the context of unstructured data; this allows the system to detect sensitive information hidden in paraphrased summaries or translated text that traditional pattern-matching tools would miss.
